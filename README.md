# OS.JS ----> Symbian OS

My quest is to literally revive Symbian OS- the operating system that came before android

Thanks Nokia, but i'll take it from here...

## Requirements

Node 8 or above

## Installation

```
# Install dependencies
npm install
```

## Development

```]

# Discover installed packages
npm run package:discover

# Build
npm run build

# Start server
npm run serve
```
